# Meu Armário 👗👕

Aplicativo para gerenciamento e catálogo de guarda-roupa pessoal, desenvolvido em **Flutter**.

## 🚀 Funcionalidades
* **Home:** Visão geral do armário.
* **Cadastro:** Adição de novas peças com detalhes.
* **Inventário:** Listagem completa dos itens cadastrados.
* **Edição:** Peças já criadas podem ser editadas.
* **Salvamento:** O app pode ser fechado que as roupas vão estar salvas.


## ✅ Em desenvolvimento:
* **Fotos:** Adicionar a opção de colocar foto da peça de roupa.
* **Classificação:** Poder classificar as roupas (Camisas,Casacos,Calças,etc).
* **Filtro:** A partir das Classificações, filtrar a pagida de inventário para mostrar apenas as peças desejadas.


## 🛠️ Tecnologias
* **Linguagem:** Dart
* **Framework:** Flutter
* **Arquitetura:** Organizada em folders de `screens`, `models`, `services` e `widgets`.
